# Hypotenuse Calculator

A simple Java program that calculates the hypotenuse of a right triangle using the Pythagorean theorem.

Users input the values of two sides (x and y), and the program computes the hypotenuse using the square root formula.

## Formula

c = √(x² + y²)

## Features

- Accepts two side values from the user
- Calculates the hypotenuse
- Displays the final result
