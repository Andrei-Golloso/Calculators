# BMI Calculator

A simple Java console program that calculates Body Mass Index (BMI) based on the user's weight and height.

The program accepts the user's weight in kilograms and height in meters, then calculates the BMI value using the standard BMI formula. It also displays the corresponding weight category.

## Formula

BMI = weight (kg) / height² (m)

## Features

- Accepts user weight and height
- Calculates BMI value
- Displays BMI category
- Console-based user input

## BMI Categories

- Underweight
- Normal weight
- Overweight
- Obese
